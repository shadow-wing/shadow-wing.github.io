Python Django从头开始

1.ubuntu安装Python3、pip3、mysql、env、django1.8.2、pycharm

  1. Python3
    1. sudo apt-get install python3
    2. which python3
    3. /usr/bin/python3
  2. pip3
    1. sudo apt-get install python3-pip
    2. pip3 -V
  3. mysql
    1. sudo apt-get install mysql-server
    2. /usr/bin/mysql
  4. env
    1. sudo pip3 install virtualenv #安装虚拟环境
    2. sudo pip3 install virtualenvwrapper #安装虚拟环境扩展包
    3. 编辑家目录下面的.bashrc文件，添加下面两行
      1. export WIRKON_HOME=$HOME/.virtualenvs
      2. source /usr/local/bin/virtualenvwrapper.sh
    4. 创建虚拟环境
      1. mkvirtualenv -p python3 虚拟环境名
        1. mkvirtualenv -p python3 dj3
      2. 进入虚拟环境
        1. workon 虚拟环境名
      3. 退出虚拟环境
        1. deactivate
      4. 删除虚拟环境
        1. rmvirtualenv 虚拟环境名
  5. django1.8.2
    1. workon 虚拟环境名
    2. pip install django==1.8.2
  6. pycharm
    1. 下载安装包
    2. 解压缩
    3. mv /opt/
    4. vi pycharm.desktop
    5. mv /usr/bin/applications
